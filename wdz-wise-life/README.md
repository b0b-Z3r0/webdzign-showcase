# WDZ Wise-Life

**Personal product — iOS app, in development and running on iPhone.**

A voice-first AI life advisor with a full voice loop: you talk, it listens, the AI processes, and it replies by voice. Tasks, habits, decisions, and guidance — hands-free.

**Stack:** React Native / Expo SDK 54 · Supabase · Claude API · Apple Speech (on-device STT)

## What it does

- Full voice loop: talk → listen → AI processing → voice reply
- Apple Sign-In and offline-first sync
- Privacy-first: raw audio is deleted after transcription, no logs
- On-device Apple Speech-to-Text
- Future direction: smart glasses / wearable hardware integration
