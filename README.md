# WebDzign — Product Showcase

Screenshots and demos of commercial software built by [Joel Munoz / WebDzign](https://www.webdzign.com), Bacoor, Cavite, Philippines.

**Why no code?** These are proprietary products running in real businesses. The source lives in private repositories; this repo shows what the products do. I'm happy to give a live walkthrough of any of them — [get in touch](https://www.webdzign.com/contact).

> All screenshots use **demo data only**. No real patient records, client names, or business data appear anywhere in this repository.
