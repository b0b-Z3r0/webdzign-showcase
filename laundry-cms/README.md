# WebDzign Laundry CMS

**Commercial product — active development.**

Offline shop management for laundry businesses, modeled after the Dental CMS architecture. Built for fast counter workflows and reliable client tracking.

**Stack:** Electron · SQLite (better-sqlite3)

## What it does

- Fast counter workflow: order in, ticket out
- Thermal receipt printing
- SMS-ready pickup messages
- Smart client deduplication by phone number (local and international formats)
- Destructive actions gated behind typed confirmation (no accidental data loss)
